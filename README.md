# Weather ETL Pipeline

This project extracts weather data from Open-Meteo API, transforms it using pandas, and loads it into PostgreSQL.

## Tech Stack
- Python
- PostgreSQL
- Pandas
- Requests

## Features
- Real-time data extraction
- Data transformation
- Database storage

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run ETL:
   python etl.py
